# Vim - Geral

## Sair

| Comando | Função                                                |
| ------- | ----------------------------------------------------- |
| `:q`    | Sair                                                  |
| `:q!`   | Forçar saída (descarta alterações não salvas)         |
| `:qa`   | Sair de tudo                                          |
| `:qa!`  | Forçar saída de tudo (descarta alterações não salvas) |
